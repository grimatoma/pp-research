Pioneer Isles

A Godot 4 / GDScript clone of Paragon Pioneers 2 — the idle, Anno-like island settlement builder. System parity with PP2 is the goal, not engine parity. Built for the aiDev portfolio; art generated with PixelLab.

The build, right now

A demo settlement on a seeded island — autotiled coastline, production buildings clustered in range of the coastal Kontor, houses populated, settlers walking between them. Rendered in-engine.

What it IS — built & validated

A playable temperate-region vertical slice covering PP2 milestones M1–M5, gated by 34 headless checks (tests/run_economy_tests.gd, exit 0/1).

Economy core done

  • The luxury cascade — each tier's luxuries are the next tier's basics, same good & per-resident rate. Verbatim PP2 denominators; golden-asserted.
  • Production chains — ~36 buildings, single-output recipes with PP2 iteration times & input ratios.
  • Population — grow on met basics, emigrate when starved (never die), Coin/Favor payout scaled by luxury satisfaction, in-place tier ascension.

World & logistics done

  • Procedural islands — seeded grass/beach/ocean/forest/mountain/river; deterministic.
  • Placement rules — forest for lumberjacks, mountains for mines, river spots for mills, coast for the Kontor.
  • Range logistics — buildings work only within Chebyshev range of a storage building. No roads, no worker assignment.

Idle & persistence done

  • Deterministic tick — advance(a)+advance(b) == advance(a+b); the remainder carries across calls.
  • Offline catch-up — bounded coarse-step replay of wall-clock time on load (no startup freeze).
  • Save/load — versioned JSON, discard-on-mismatch, mid-cycle accumulators preserved.

Architecture

Follows the spec's directive: const catalogs + one mutable world object, not many singletons — which also makes the whole sim reachable headless for tests.

Per-tick order (fixed, golden-asserted): connectivity → production → consumption → population → payout.
